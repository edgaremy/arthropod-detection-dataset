## OOD dataset requirement

The Python script in this folder requires the OOD dataset from Zenodo to be downloaded and extracted in this same folder to work properly:

- URL: https://zenodo.org/records/15096610
- DOI: 10.5281/zenodo.15096609

Expected location after extraction:

- `datasets(others)/OOD/`

If the dataset is not extracted here, dataset build/processing scripts in this folder may fail because required files will be missing.
